# Scholars Hub
Scholars Hub is a web-based study session management system built for students to create, join, and manage academic study groups within their year group. The platform enforces cohort-based access, meaning students only see and interact with sessions relevant to their registered year. Users can register with an institutional email, log in securely, create study sessions with details such as course, time, location, and capacity, and join or leave sessions as needed. Session creators have additional controls such as cancelling sessions or removing participants. The project is built using PHP for backend logic, MySQL for data storage, and HTML/CSS with JavaScript for the frontend, and is designed to run locally for testing with XAMPP.

# Setting up
To set up this project, download the repository into your local XAMPP htdocs folder, then start Apache and MySQL from XAMPP. Import the database/study_platform.sql file into phpMyAdmin to create the required database and tables. Update the database connection details in config.php or config/database.php if necessary. Finally, open your browser and run the project.
